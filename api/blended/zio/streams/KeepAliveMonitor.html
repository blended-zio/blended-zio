KeepAliveMonitor

trait KeepAliveMonitor extends AnyRef

A keep alive monitor that that expects a regular keep alive signal. Every time the signal is missed, a counter is increased up to a given maximum.

The run effect executes until the given maximum is reached and then terminates with the current count of missed keep alives.

The instance of the monitor will die once it has been triggered and a new instance should be created by users of the monitor once they have taken whatever measures they require to recover from the missed keep alives.
